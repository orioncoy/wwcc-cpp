# Midterm

## Building

### Setting up the Makefile

Make sure to:
 - `CXX`: set to a compiler you have installed
 - `CXXFLAGS`: add any flags you deem necessary under

[!IMPORTANT]
This program uses C++17, change this at your own risk.
```
# Compiler and Flags
CXX = g++
CXXFLAGS = -Wall -std=c++17
```
